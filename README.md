# Logfox Homebrew Tap

Homebrew formulae for [Logfox](https://logfox.ai) tools.

## Installation

```bash
brew tap logfoxai/tap
```

## Available Formulae

### open-prs

TUI + CLI dashboard for all open PRs across a GitHub org.

```bash
brew install logfoxai/tap/open-prs
```

Or install directly without tapping first:

```bash
brew install logfoxai/tap/open-prs
```

[View on GitHub](https://github.com/logfoxai/open-prs)
